# Formday

**Formday** is a modern, disciplined fitness tracking application built to help users seamlessly manage their workouts, meals, and progress. Designed with a calm and motivating aesthetic, it combines powerful tracking features with a premium user experience.

## Features

- **Daily Snapshot**: Instant overview of your daily workout and meal progress.
- **Workout Planner**: Flexible tools to schedule and track your resistance training and exercises.
- **Meal Planner**: Simple and effective nutritional tracking to keep your diet on point.
- **Analytics Dashboard**: Visual insights into your consistency and performance over time.
- **Responsive Design**: A fluid, mobile-first interface optimized for any device.

## Tech Stack

Built with the [T3 Stack](https://create.t3.gg/):

- **Framework**: [Next.js](https://nextjs.org) (App Router)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com) & [Radix UI](https://www.radix-ui.com/)
- **State & Data**: [tRPC](https://trpc.io) & [TanStack Query](https://tanstack.com/query)
- **Database**: [Prisma](https://prisma.io) & SQLite
- **Animations**: [Framer Motion](https://www.framer.com/motion/)

## Getting Started

1. **Install dependencies**:

   ```bash
   bun install
   ```

2. **Set up the database**:

   ```bash
   bun db:push
   ```

3. **Run the development server**:
   ```bash
   bun dev
   ```

Open [http://localhost:3000](http://localhost:3000) to view the application.
